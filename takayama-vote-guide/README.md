# 高山市長選挙 投票マッチング

「誰に投票したらいいか分からない」という方が、かんたんなアンケート（全12問）に
答えるだけで、**自分の考えに近い候補者**を参考表示できる Web サイトです。

- 回答はすべてブラウザ内で処理され、外部に送信されません。
- 5段階（賛成〜反対）で回答し、重視するテーマには重み付けできます。
- 質問ごとに「あなた vs 各候補者」の比較表も表示します。

## ⚠ 重要：候補者データはサンプル（架空）です

`data.js` に入っている候補者A・B・Cは **架空のサンプル** です。実在の候補者の
主張ではありません。実際の選挙で利用する場合は、必ず以下の **一次情報** にもとづいて
`data.js` の `CANDIDATES` を更新してください。

- 選挙公報
- 各候補者の公式サイト・公約・政策ビラ
- 公開討論会・アンケートでの本人の発言

出典が確認できない主張を候補者に紐づけて公開することは避けてください。

## データの編集方法

`data.js` だけを編集すれば設問と候補者を差し替えられます。

```js
// 設問を追加・修正
const QUESTIONS = [
  { id: "tourism", category: "観光・産業", text: "…", detail: "…" },
  // …
];

// 候補者ごとに、各設問 id に対する立場を -2〜+2 で指定
const CANDIDATES = [
  {
    id: "A",
    name: "○○ ○○",
    catchphrase: "キャッチコピー",
    color: "#d9534f",
    summary: "候補者の特徴の要約",
    stances: { tourism: 2, residents_first: -1, /* …全設問… */ },
  },
];
```

立場の値: `+2`=強く賛成 / `+1`=やや賛成 / `0`=中立 / `-1`=やや反対 / `-2`=強く反対

## ローカルでの確認

ビルド不要の静的サイトです。

```bash
cd takayama-vote-guide
python3 -m http.server 8000
# ブラウザで http://localhost:8000 を開く
```

## 公開（デプロイ）

`index.html` / `styles.css` / `app.js` / `data.js` を任意の静的ホスティング
（Vercel, GitHub Pages, Netlify など）に置くだけで動きます。Vercel の場合は
このディレクトリを「Root Directory」に指定してください。

## 構成

| ファイル | 役割 |
| --- | --- |
| `index.html` | 画面（スタート／設問／結果） |
| `styles.css` | デザイン |
| `app.js` | アンケート進行とマッチング計算 |
| `data.js` | 設問・候補者データ（ここを編集） |

## 免責

本サイトはあなたの回答とデータの一致度を示す参考ツールであり、特定の候補者への
投票を推奨するものではありません。投票の判断はご自身の責任で、公式情報を
ご確認のうえ行ってください。
