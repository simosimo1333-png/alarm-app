# 飛騨高山ファイター 🏯👊

飛騨高山をモデルにした2D対戦格闘ゲームです。
**古い町並み（さんまち）・高山祭・造り酒屋・飛騨の里**をテーマにした
ステージと、飛騨ならではのキャラクターで対戦できます。

[Phaser 3](https://phaser.io/) 製の**ビルド不要の静的サイト**なので、
そのまま Vercel・GitHub Pages・ローカルサーバーで動きます。

---

## 🎮 遊び方

### 📱 スマホ・タブレット（タッチ）
タッチ端末では画面下に操作ボタンが自動表示されます（Player 1 を操作）。

- 左下：**◀ ▶**（移動）／ **▲**（ジャンプ）／ **▼**（しゃがみ）
- 右下：**弱**（パンチ）／ **強**（キック）／ **必殺**
- ガード：相手と**反対方向**の移動ボタンを押し続ける
- 中央下：**II**（一時停止）
- 移動と攻撃の**同時押し**に対応（マルチタッチ）
- スマホでは **1人プレイ（vs CPU）** がおすすめです。

### ⌨️ PC（キーボード）

| 操作 | Player 1 | Player 2 (2人対戦時) |
|------|----------|----------------------|
| 移動 | `A` / `D` | `←` / `→` |
| ジャンプ | `W` | `↑` |
| しゃがみ | `S` | `↓` |
| 弱攻撃（パンチ） | `F` | `J` |
| 強攻撃（キック） | `G` | `K` |
| 必殺技 | `H` | `L` |
| ガード | 相手と**反対方向**に移動入力 | 同左 |
| 一時停止 | `P` | — |

- **2本先取**でラウンド勝利（ベスト・オブ・3）。
- 体力をすべて削るか、制限時間（60秒）終了時に体力が多い方が勝ち。
- 連続ヒットで **HITコンボ** が表示されます。

## 🚀 ローカルで動かす

`index.html` を開くだけでも動きますが、`file://` だと一部ブラウザで
読み込み制限があるため、簡易サーバー経由を推奨します。

```bash
# Python があれば
python3 -m http.server 8000
# → http://localhost:8000 を開く

# もしくは Node
npx serve .
```

## ▲ Vercel で公開する

このリポジトリはフレームワーク不要の静的サイトです。

1. [vercel.com](https://vercel.com/) にGitHubアカウントでログイン
2. **Add New… → Project** から本リポジトリ `hida-takayama-fighter` をインポート
3. **Framework Preset** は `Other`（自動検出のまま）、ビルド設定は空でOK
4. **Deploy** を押すだけで公開URLが発行されます

> 設定不要で `index.html` がそのまま配信されます（`vercel.json` で
> キャッシュ設定のみ付与しています）。
