# 🏁 Animal Kart Racing / アニマル カートレーシング

ブラウザだけで遊べる、擬似3D（OutRun方式の道路投影）のカートレーシングゲームです。
画像アセットは一切使わず、カート・キャラクター・背景はすべて Canvas で手描き描画しています。

添付のリファレンス画像（クラッシュ／ソニックレーシング風）を参考に、以下の3つのテーマコースと
HUDデザインを再現しています。

## 🎮 遊び方

`kart-racer/index.html` をブラウザで開くだけ。サーバー不要・ビルド不要。

| 操作 | キーボード | タッチ |
|------|-----------|--------|
| アクセル | ↑ / W | GO ボタン |
| ブレーキ／バック | ↓ / S | ⬇ ボタン |
| ハンドル | ← → / A D | ◀ ▶ ボタン |
| アイテム使用 | Space | 🎁 ボタン |
| ドリフト | Shift（旋回中） | — |

ドリフトを一定時間続けてから離すと、ミニブースト（青い炎🔥）が発動します。

## 🗺 コース（リファレンス画像に対応）

1. **ジャングル神殿** 🌴 — 緑の渓谷・滝・木の道（画像1）
2. **ネオンシティ** 🌃 — 夜景・ネオンの看板・紫＆シアンの光（画像2）
3. **トロピカルビーチ** 🏝 — 青空・海・遠くの城（画像3）

## 🧩 HUD（画像のレイアウトを再現）

- 左上：LAP カウンター＋ラップタイム
- 左：順位リスト（自分はオレンジでハイライト）
- 右上：大きな順位表示（`1st` / `4th` …）
- 右上：ミニマップ（全車をドット表示）
- 左下：アイテムボックス
- 右下：スピードメーター（シアン→オレンジのゲージ＋ km/h）

## 🛠 アイテム

| アイコン | 効果 |
|----------|------|
| 🔥 ブースト | 短時間の加速 |
| ⚡ 雷 | 強めの加速 |
| 🛡 シールド | 一定時間ダメージ無効 |
| 🚀 ロケット | 前方の相手をスピンさせる |

## ⚙ 技術メモ

- 純粋な HTML + CSS + JavaScript（依存ライブラリなし）
- 擬似3D道路：セグメント分割＋透視投影（カーブ・起伏・霧）
- AI はラバーバンド付きでコースを周回
- 6台（プレイヤー1＋AI5）で3周レース
- PC・スマホ（タッチUI）両対応・レスポンシブ
